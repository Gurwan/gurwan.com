Soundbox France

Personal project | October 2017 - May 2021

Application created in Java on Android Studio with AndroidX and SQLite libraries

This project is my first programming project and I partially learned to code with this project. Of course the project being more than three years old, I improved it as I progressed in the field. The primary objective of this project was to put it into production, so Soundbox France has been available since autumn 2017 on the Google Play Store platform. The application has had some success with in particular 380,000 unique users having downloaded the application. This project allowed me to discover a large part of the phases of creating a mobile application.

Link to the Play Store and to the Github.

Resolution Held

Personal project | December 2020 - January 2021

Application made in Kotlin with in particular the Gson library

Every year, most people make a resolution list to improve their lives. However, the main difficulty of this custom is not to choose these resolutions but to keep them. To fix this problem: Resolution Held. This mobile application makes it possible to hold 2 resolutions throughout the year. Each day is represented by a circle which can be clicked on when the resolution is held. However, the objective is not over a year but over a month. Indeed, each resolution lasts a month. At the end of each month you will be able to restart the challenge, throughout the year.

Link to the Github.

Zen l'Initié

IUT project | April 2020 - June 2020

Application created in Java with in particular the Swing graphics library

This project is the summary project of my first year of DUT Informatique. The objective is to build an entire game using the Java language. The game is a digital adaptation of the board game Zen the Initiate. The goal of Zen the Initiate is to succeed in forming a continuous chain with all of his pawns still on the board. If the red pawn is still on the board then it must also be aligned with its pawns. The game can take place over several turns according to your choice. This computerized board game is built in a terminal version, but also in a graphical version with the Swing graphics library.

Link to the Github.
