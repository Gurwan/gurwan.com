Soundbox France

Projet personnel | Octobre 2017 - Mai 2021

Application réalisée en Java sur Android Studio avec les bibliothèques AndroidX et SQLite

Ce projet est mon premier projet de programmation et j'ai en partie appris à coder avec ce projet. Bien entendu le projet ayant plus de trois ans, je l'ai amélioré au fur et à mesure de ma progression dans le domaine. L'objectif premier de ce projet était de le mettre en production, Soundbox France est donc disponible depuis l'automne 2017 sur la plateforme Google Play Store. L'application a connu un certain succès avec notamment 380 000 utilisateurs uniques ayant téléchargé l'application. Ce projet m'a permis de découvrir une grande partie des phases de création d'une application mobile.

Addicted Stalked

Projet personnel | Décembre 2022-...

Application réalisée en Java sur Android Studio

Addicted Stalker est un projet d'application mobile visant à contrer les algorithmes dérégulant votre dopamine pour vous maintenir prisonnier de votre smartphone. L'objectif de cette application est de vous faire subir des "électrochocs" destinés à vous sortir de la boucle grâce à des notifications incessantes et imparables. La solution pour voir ces notifications vous laisser enfin tranquille ? Régulez votre consommation et vivez votre vie en dehors de votre flux vidéo quotidien. Autre solution : désinstaller l'application et accepter de vivre une vie numérique et sans but.

Resolution Held

Projet personnel | Décembre 2020 - Janvier 2021

Application réalisée en Kotlin avec notamment la bibliothèque Gson

Chaque année, la plupart des gens font une liste de résolution afin d'améliorer leur vie. Cependant la principale difficulté de cette coutume n'est pas de choisir ces résolutions mais de les tenir. Pour régler ce problème : Resolution Held. Cette application mobile permet de tenir 2 résolutions tout le long de l'année. Chaque jour est représenté par un rond sur lequel il suffit de cliquer lorsque la résolution est tenue. L'objectif n'est pas cependant sur une année mais sur un mois. En effet, chaque résolution dure un mois. À la fin de chaque mois vous pourrez redémarrer le challenge, et cela, pendant toute l'année.

Zen l'Initié

Projet IUT | Avril 2020 - Juin 2020

Application réalisée en Java avec notamment la bibliothèque graphique Swing

Ce projet est le projet de synthèse de ma première année de DUT Informatique. L'objectif est de construire un jeu entier avec le langage Java. Le jeu est une adaptation numérique du jeu de société Zen l'Initié. Le but de Zen l'Initié est de réussir à former une chaîne continue avec l’ensemble de ses pions se trouvant encore sur le plateau. Si le pion rouge est encore sur le plateau alors il faut également l’aligner avec ses pions. La partie peut se dérouler sur plusieurs tours selon votre choix. Ce jeu de plateau informatisé est construit dans une version terminale, mais également dans une version graphique avec la bibliothèque graphique Swing.
